R PROGRAMMING LANGUAGE

WHAT IS R?

R is a programming language developed by Ross Ihaka and Robert Gentlemen in 1993 and is free software. R has an comprehensive mathematical and graphical process catalog. It contains algorithms for machine learning, linear regression, time series, to name a few, statistical inference. The R libraries are mainly written in R, but C, C++ and Fortran codes are preferred for heavy computational tasks.Not only is R entrusted to academia, but several major corporations, including Uber, Google, Airbnb, Facebook and so on, still use R programming language.

WHAT R IS USED FOR?

-Statistical inference
-Data analysis
-Machine learning algorithm

SHOULD YOU USE R?

Two excellent methods can be used by data scientists: R and Python. You might not have time to learn both of them, especially if you start studying data science. It's much more important to study mathematical modeling and algorithms than to learn a programming language. A programming language is an instrument for the measurement and correspondence of your exploration. In data science, the most important activity is how you work with the data: importing, washing, planning, feature engineering, selecting features.
Market analysts are the main audience for data science. One main implication of the industry is connectivity. There are many ways of interacting: log, mobile app, dashboard. You need a tool to do all these things together.
